## ItemSign

ItemRename is a simple plugin that allows you to add a signature to an item

## Configuration

You can change the maximum length, the usage of colors, default signature color and the default signature message in `plugin_data/ItemRename/config.yml`

## Commands

| Default command | Parameter | Description | Permission |
| :-----: | :--------: | :---------: | :----------: |
| /itemsign | `<message>` | Signs the item you're holding | command.itemsign |
| /itemclearsign | --- | Clears the signature of an item | command.clearitemsign |
